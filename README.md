# Apomorphine for ED

An educational, single-ingredient website about apomorphine and its dopaminergic
role in treating erectile dysfunction.

- Static site: `index.html` (home) + three sections, each a folder with its own page:
  `apomorphine-dopamine/` (mechanism), `ranked-apomorphine/` (treatments), `apomorphine-ed/` (evidence).
- Shared assets in `assets/` (`styles.css`, `app.js`, hand-authored SVGs in `assets/img/`).
- Content is cited; not medical advice.

Serve from the repo root, e.g. `python3 -m http.server`.
